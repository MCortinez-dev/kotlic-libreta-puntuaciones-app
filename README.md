# Kotlin Score Notebook App

This project is a score tracking application developed in Kotlin.

The application allows users to keep track of scores for games or competitions using a simple interface.

## Features

- Add players
- Track scores
- Update results
- Simple score management

## Technologies

- Kotlin
- JVM
- Gradle

## Purpose

This project was created as part of a programming assignment to practice application logic and Kotlin development.

## How to Run

1. Clone the repository

git clone https://github.com/MCortinez-dev/kotlic-libreta-puntuaciones-app

2. Open the project in IntelliJ IDEA or another Kotlin compatible IDE

3. Run the main application file
